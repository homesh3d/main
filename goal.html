Design Goal


❏ General Purpose

    Grid generated by HOMESH can be applied to the general field of computational mechanics including,but not limited to, CFD, CSM, CGM, …etc.

    The data structures representing a grid system can be used by various numerical methods including, but not limited to, FDM, FVM, FEM, …etc.


❏ Grid Generation Package

    Integration of structure grid and unstructured grid generation methods,

    Integration of single module, multi-region, multi-block topological concepts.
